==============
Operations API
==============


Overview
--------

API for generating and manage UI forms for reclass model templates

Requirements
------------

-  Python v3.6 and higher.
-  Pip v3 and higher.

Installation notes
------------------

- Prepare python virtual environment

  ::

    python -m ensurepip --default-pip
    pip install --user pipenv
    pipenv --python 3.6
    pipenv install
    pipenv shell

    python operations_api
